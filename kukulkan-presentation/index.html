¿Qué es Kukulkan?

Kukulkan es una herramienta de software hecha para los practicantes de ingeniería de software que ayuda a incrementar la productividad de proyectos de software a través de la gestión de componentes reutilizables y herramientas para la generación de código.

Generador de Codigo

Kukulkan posee un motor para la generación a partir de un modelo de datos descrito en varias tecnologías (MySQL, PostgreSQL, Oracle, SQLServer, etc).


Administrador de Componentes

Kukulkan es una herramienta para la administración de componentes reutilizables organizacionales.

Modelador de Entidades

Kukulkan tiene una gramática para la descripción de entidades, a partir de las cuales puede generar código para varias tecnologías (Java, JavaScript y .Net)


¿Nuevo en una tecnología?

No hay problema, a través de kukulkan puedes ir explorando mejores maneras de programar y desarrollar un generador que permita reutilizar tu estrategia.
